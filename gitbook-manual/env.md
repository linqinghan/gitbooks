# 安装环境

## 安装node.js

通过一下确认安装成功

> node -v 

> npm -v

## 安装gitbook 

> npm config set proxy http://192.168.1.238:808    --> 设置http代理

> npm config set https-proxy http://192.168.1.238:808  --> 设置https代理

> npm install -g gitbook-cli

> npm install -g gitbook 

确认安装成功：

> gitbook -V

卸载

> npm uninstall -g gitbook
